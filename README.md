# SenTinMixt

The file SenTinMixt_1.0.0.tar.gz contains the R package for fitting the models of Tomarchio, S.D., Bagnato, L., & Punzo, A. Model-based clustering via new parsimonious mixtures of heavy-tailed distributions. AStA Adv Stat Anal 106, 315–347 (2022). https://doi.org/10.1007/s10182-021-00430-8 
A help description for each argument supports each function and dataset within the package.
